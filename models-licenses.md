# Models & Licenses — guidance

This file provides guidance and links for checking licenses of common models used with Joker Smile.

- FLUX.1
  - Model card / source: (please add official link here)
  - Notes: Check the model card for commercial use restrictions, attribution, and terms.

- Stable Diffusion XL (SDXL)
  - Common model cards: check Hugging Face / StabilityAI pages for the specific checkpoint you plan to use.
  - Notes: Many checkpoints have specific restrictions; do NOT commit weights to this repo.

Guidance:
- Always read the model card on the hosting site (Hugging Face, official repo) before using the model for commercial purposes.
- Do not commit checkpoints/weights into this repository. Provide instructions for users how to download from the official host.
- If you include third-party assets (fonts, images, templates), ensure their license allows the intended use and provide attribution when required.
