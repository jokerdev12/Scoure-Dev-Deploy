# Contributing to Joker Smile

Cảm ơn bạn đã quan tâm đóng góp! Các bước cơ bản:

1. Mở issue trước khi làm PR lớn để thảo luận ý tưởng/thiết kế.
2. Fork repo, tạo branch tính năng, và gửi PR với mô tả rõ ràng.
3. Tuân thủ style code (Python: black/isort; JS: prettier) nếu có.
4. Bao gồm test cho hành vi mới khi hợp lý.
5. Không commit model weights, API keys, secrets hoặc tài nguyên có license hạn chế.
6. Tôn trọng cộng đồng: giữ lời lẽ lịch sự và hợp tác.

Nếu cần hỗ trợ onboarding, mở issue với tiêu đề "help wanted" và maintainer sẽ trợ giúp.
